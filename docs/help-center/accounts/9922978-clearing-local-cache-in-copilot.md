# Clearing Local Cache in Copilot

**Source:** https://help.copilot.money/en/articles/9922978-clearing-local-cache-in-copilot

Clearing the local cache helps to remove any stale data that's been saved in Copilot and load refreshed data to your device. There are several reasons why you might want to clear the local cache in Copilot. We will share some examples below as a guideline on when clearing the local cache in Copilot can be helpful.

---

# Duplicate Transactions

Duplicate transactions can be caused by a data caching issue, rather than connectivity issues. When looking at these duplicate transactions, if one copy of the transaction is associated with an account but the other copy is not, this is most likely a data caching issue. This is usually caused by deleting and reconnecting the same account, but the transactions were not removed properly from the deleted copy.

You can clear the local cache in Copilot by navigating to **Copilot Settings > Advanced > Clear the local cache**or **Copilot Settings > Account > Clear local cache** on Web. Give it a few seconds. If this does not remove the duplicate transactions, please reach out to the Copilot team via our in-app chat with any relevant screenshots so we can troubleshoot further.

# Missing Transactions

Sometimes, the transaction data for your Copilot account is not properly loaded on to your device, causing missing transactions or inaccurate budget and spending data that was accurate before.

If this is the case, you can follow the steps above to clear the local cache in Copilot. If the issue persist, please reach out to the Copilot team via our in-app chat with any relevant screenshots so we can troubleshoot further.

# Pending and Posted Transactions

When a posted copy of a transaction surfaces in Copilot, the pending copy should be recalled by the data aggregator. If you're seeing both pending and posted copy of the transaction, we recommend that you force close and re-open the app first to check if the pending copy of the transaction is removed. If not, please follow the steps above to clear the local cache in Copilot.

If you are still seeing the pending copy of the transaction after, please contact the Copilot team via our in-app chat with any relevant screenshots so we can troubleshoot further.

# App Performance

If you're experiencing app performance issues, or if the Copilot app is taking up an unusual amount of storage on your device, please follow the steps above to clear the local cache in Copilot.

If you're still experiencing the same issue, please contact the Copilot team via our in-app chat and share more details on the app performance issue you're experiencing and we can help troubleshoot further. Thank you!
