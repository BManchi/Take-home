# Understanding Manual Accounts

**Source:** https://help.copilot.money/en/articles/10682991-understanding-manual-accounts

# Balances on Manual Accounts

You can choose to create manual accounts for Credit Cards, Checking and Savings (Depository), Investments, Loans, Real Estate, and Cash. Please note that Checking and Savings (Depository) and Cash accounts do not support negative balances.

## Historic Balances

When you [add a manual account](https://help.copilot.money/en/articles/4537532-creating-manual-accounts), you'll see balance data starting from the date of creation. At this time, Copilot does not support creating historic data points or importing historic balance data.

## Editing the Balance

Copilot allows you to edit today's balance. To do this, please tap the manual account > tap the current balance > Edit > SAVE.

# Manual Transactions on Manual Accounts

Manual transactions are supported for the following manual account types:

- Credit Cards
- Checking and Savings (Depository)
- Loans
- Cash

## Transaction Dates

When you [create a transaction](https://app.intercom.com/a/apps/gw2wbwl7/knowledge-hub/all-content?activeContentId=3383268&activeContentType=article&editorMode=view&search=manual%20transaction&types=120) for today's date, your manual account's balance will update automatically. Deleting that transaction will also update the account balance.

## Historic Transactions

When you create a transaction for a date in the past, the balance will change according to the account type:

- For **Loans**, historic transactions do not update today's balance.
- For **Credit Cards** and **Checking and Savings (Depository)**, transactions are expected to update today’s balance. However, historic transactions from a date prior to when you created the account in Copilot will not update today’s balance.
- For **Cash** accounts, all historic transactions are expected to update today's balance.

If creating a historic transaction causes an incorrect account balance, please manually edit today's balance.

## Future-Dated Transactions

When you create a transaction for a date in the future, today's balance will automatically update. This is true for all account types.

If creating a future-dated transaction causes an incorrect account balance, please manually edit the balance.
