# Groups of Categories

**Source:** https://help.copilot.money/en/articles/3767655-groups-of-categories

Copilot allows you to group categories together to better organize your spending.

---

# Create a Group of Categories:

- From the **Categories** tab, select an existing category, then tap the menu button at the top right.
[![screenshot](../../screenshots/categories/01-Pic01.png)](https://downloads.intercomcdn.com/i/o/525516654/324e3430f17d8add8cae267b/Pic01.png?expires=1773322200&signature=80136521bf86b6cb5c65294693b3fb7f8a850930d882c407f96cadf6befe5b18&req=cSIiE8h4m4RbFb4f3HP0gKfy4otj8yNW5K69STPAk%2BEZA9MltcOMawm5tSCO%0AGZcksYOl3RRQoJF2TQ%3D%3D%0A)
- From the **Category Settings** menu, select the **Spending category group** option.
[![screenshot](../../screenshots/categories/02-Pic02.png)](https://downloads.intercomcdn.com/i/o/525516755/d8d9b9fd4e38f47add80f8c1/Pic02.png?expires=1773322200&signature=644513531ad62dad48e1bffeb09ff5269d602bcfaf9257e31eb200364f8dedbd&req=cSIiE8h4moRaFb4f3HP0gJLTGkdTbYII6Sj3D1s5HVV60hXgJivKdO88xFOr%0AjukGiHHqC9ptH1YVwQ%3D%3D%0A)
- In the **Groups of Budgets** menu you can select an existing group or **Start a new group.**
[![screenshot](../../screenshots/categories/03-Pic03.png)](https://downloads.intercomcdn.com/i/o/525516810/091f9c83f6af3309e5ea17e2/Pic03.png?expires=1773322200&signature=6b1e900f8aee551b62af63f5d68b7b2471cbc4de01f650640c90d2f169aa7cb1&req=cSIiE8h4lYBfFb4f3HP0gIAo69B9Cg0gvOaOcJ%2F5cd30ymS5otC0w%2BoOSFjF%0Ai14AsR3jbR7Aiha6PQ%3D%3D%0A)
- Select **Start a new group**and enter the name. You can also enter an **Unassigned Budget** for the group. Set an unassigned budget if you do not want to set a budget for each subcategory or if you want to add budget for subcategory overages.

*Example: Create the Group “Food” with categories “Groceries” and “Restaurants”, and set the Group unassigned budget to $500 rather than $250 for Groceries and $250 for Restaurants. This is useful if your monthly spend fluctuates due to cooking vs. dining out more often.*

- Save and then select another category to add to the group.
[![screenshot](../../screenshots/categories/04-Pic04.png)](https://downloads.intercomcdn.com/i/o/525516906/d8df455cf9e83725525c130b/Pic04.png?expires=1773322200&signature=64ebbe7e6504e0100dc88d949cf161c0ada9d84ecda1cde85d429e3159093bab&req=cSIiE8h4lIFZFb4f3HP0gEkHzjmGWe1a%2BagwXsql7MUYpCkfDWgSmQJZEf2a%0APzNLMiHrrmq2tGqY8A%3D%3D%0A)
- Open the menu on the top right and select **Spending category group** again and choose your recently created group name.
[![screenshot](../../screenshots/categories/05-Pic05.png)](https://downloads.intercomcdn.com/i/o/525516953/d92bd95b40e386d48f8c0745/Pic05.png?expires=1773322200&signature=7fe03d12fae0439a23138a5af15ad6d3c473b10a3b684569c1be4f299a68a171&req=cSIiE8h4lIRcFb4f3HP0gPVfazMPZrc6%2FDKJOeQdxGBYdVSaBuoJtzhwaMzc%0AeX%2BWD3pBxDoekk7nvQ%3D%3D%0A)
- Then on the **Categories** tab, you will see the new group which you can expand or select to see the group budget and subcategories.
[![screenshot](../../screenshots/categories/06-Pic06.png)](https://downloads.intercomcdn.com/i/o/525517055/52e9fa4dd5ec67668bfeddc6/Pic06.png?expires=1773322200&signature=d8dd66846bd403f0e549919433b9e4e42dd1c25ecb512433612bfe91fb125c33&req=cSIiE8h5nYRaFb4f3HP0gKz2ycRHDXYv6iSZess7EXqIW1SxCiQtUsibpmK%2B%0A1S1CViK5QlteFzH4vA%3D%3D%0A)
# Edit Group Budgets:

- From the **Categories** tab, select an existing **Group**.
[![screenshot](../../screenshots/categories/07-Pic06.png)](https://downloads.intercomcdn.com/i/o/525517206/23902cb6585d1c4a06837331/Pic06.png?expires=1773322200&signature=4a822ddf5e60a4449bad6b6097e6b5fc26dbd4cc941653ed9f0f4570d4df0964&req=cSIiE8h5n4FZFb4f3HP0gJJL0eS578Ytreq5s21zpXtYOsV6daGhZG1bw2cn%0AOINxzA8SJN0LFH7cBQ%3D%3D%0A)
- To edit a subcategory budget, select the individual category.
[![screenshot](../../screenshots/categories/08-Pic07.png)](https://downloads.intercomcdn.com/i/o/525517262/87698bdc3c3196700c9695d1/Pic07.png?expires=1773322200&signature=a33d9d36aaa83e1fd2f1d45a641fc316a644f9023250cd8bb5d83c643033e9cc&req=cSIiE8h5n4ddFb4f3HP0gHmLv%2BFcO4XjW1w%2BMk6Akm%2FHKohOHIj%2BbIcZKjWV%0AhIFJr4yXZBlfG213iA%3D%3D%0A)
- Then, tap or slide the budget bar to adjust the budget.
[![screenshot](../../screenshots/categories/09-Pic08.png)](https://downloads.intercomcdn.com/i/o/525517315/24e78916a338b882e3202583/Pic08.png?expires=1773322200&signature=6c45a4cb62481b00741b163459806e355d433f98e38a7fe8192ba8f9bced1eaf&req=cSIiE8h5noBaFb4f3HP0gKDPggAZZZ37lKw3SVqUYlr3kiloEcAwxqMSdi58%0AVTEsrbCapZwwbjjQYg%3D%3D%0A)
- To edit a group **Unassigned Budget**, either tap on the group name or slide the budget bar to adjust the subcategory and/or unassigned budgets.
[![screenshot](../../screenshots/categories/10-Pic09.png)](https://downloads.intercomcdn.com/i/o/525517372/caed3f848d56d1a3e14c34bf/Pic09.png?expires=1773322200&signature=c82909e6218f32709bef1ce8c69fe928f3c4cc6ddea7321281b643a478b59dbe&req=cSIiE8h5noZdFb4f3HP0gNALziPWjwZOhpNmzBlREATySKKyMv1C0SAO37Nt%0An0AGCBI189gjM%2BH2tg%3D%3D%0A)
- Set an**Unassigned Budget** if you do not want to set a budget for each subcategory or if you want to add budget for subcategory overages.
[![screenshot](../../screenshots/categories/11-Pic09.png)](https://downloads.intercomcdn.com/i/o/525517637/c39f3269347ec12b0f2260f5/Pic09.png?expires=1773322200&signature=8ef2df58483fe55ae89e18bbe0cfaa89eaa1098e377b89727d62e550431a6213&req=cSIiE8h5m4JYFb4f3HP0gDjAURAFqisGEpWKvKKFGlS%2BIGoNR6xgh0OR3Lgm%0ATLcy75nCDP4S2eiUBg%3D%3D%0A)[![screenshot](../../screenshots/categories/12-Pic10.png)](https://downloads.intercomcdn.com/i/o/525517688/cc74bbfa3b1488d65c06da6f/Pic10.png?expires=1773322200&signature=3c0172c3412549fef5b3e77921b5168842a23e12f05e7618a5b8690ea964c465&req=cSIiE8h5m4lXFb4f3HP0gKHHaMFwDOYaCoRAZN79YHzdLSAIW1HgFY04Qv0I%0Azs%2Bsl%2FRHRqOSLXVj4Q%3D%3D%0A)
- In the **Groups of Categories** view, you will see the new group budget.

# Remove a Single Category from a Group:

- From the **Categories** tab, select an existing category, and then open the menu on the top right, then select **Spending category group** at the bottom of the view.
- In the **Groups of Budgets**menu, select **Remove from group.**
[![screenshot](../../screenshots/categories/13-Pic11.png)](https://downloads.intercomcdn.com/i/o/525517767/a6918d128c6c4bb6ff970069/Pic11.png?expires=1773322200&signature=9729c62851e620f7bfaaa4739b0387dfc2c5756dcd7e5943b4ecf9b8740bb819&req=cSIiE8h5modYFb4f3HP0gGWeNfG7c0V27O8duK%2FPlxgymK88dc3iCU4wAI9%2B%0AmeU9yT83zwEMV%2Ftxmw%3D%3D%0A)
# Ungroup a Set of Categories:

- From the **Categories** tab, select an existing group and from the menu on the top right select **Ungroup these categories** at the bottom of the view.
- **Confirm Ungrouping** or close the view to cancel.
[![screenshot](../../screenshots/categories/14-Pic12.png)](https://downloads.intercomcdn.com/i/o/525517833/243ef8a6f24ffaa42dc4bcc8/Pic12.png?expires=1773322200&signature=d2db9f34cf18adc32abe349076ee14c4fcde14a73c24e2f2600d6c7171e33407&req=cSIiE8h5lYJcFb4f3HP0gJH7XfSeq31K4nkUpuXJzSVRSwfmeGbrYS9xRAGY%0AnmN%2FUadK1F1%2BSYx4Nw%3D%3D%0A)
