# Group of Categories for general budgeting

You can move all of your categories into one group if obsessing over individual category spending and budget isn't your thing. This allows you to edit the overall budget for the month with ease without needing to decrease or increase the budget for individual categories.

You'll start by moving all categories into one group (you can use this [article](https://help.copilot.money/en/articles/3767655-groups-of-categories) to learn more about group of categories), then tap on the group, tap on the three dots on the upper right hand corner in the group view, tap on edit budget, tap on **unassigned** to enter the budget you'll like to set for your overall budget for the month, then set the assigned budget to $0. This will remove any existing budget set for the subcategories.

**Another example:**

You can take the same idea shared above to create more groups if you'd like.

In this example below, you'll see 2 group of categories named **Need** and **Want**.

The **Need** group includes any subcategories that are necessary spending, like Rent, Transportation, Subscriptions, etc.

The **Want** group includes any subcategory that you might spend money on, like Clothing, Restaurant, Shops, etc.

# Excluded Categories

You can use excluded categories to track spending that you don't want to include in your monthly spending totals. This can be useful for tracking transactions related to work expenses, a home renovation project, or spending for a family member without impacting your own budget and spending.

# Archived Categories

If you have categories you no longer use, you can create a group of categories named **Archived Categories**or **Archived**. This allows you to move any categories you no longer use into this group to clean up your Categories tab. You'll also want to make sure you set the budget for these categories to $0 moving forward so they no longer impact your overall budget.
