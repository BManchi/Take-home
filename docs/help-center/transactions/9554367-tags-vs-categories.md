# Tags vs. Categories

**Source:** https://help.copilot.money/en/articles/9554367-tags-vs-categories

***Categories*** are only used for categorizing regular transactions. **[Learn more about the different transaction types here](https://help.copilot.money/en/articles/3971267-transaction-types)**. Categories can have a budget associated with them, and can be grouped together under a single Category Group. Categories also have their own tab in Copilot for review and management. **[Learn more about Categories here.](https://help.copilot.money/en/collections/2199956-categories)**
[![screenshot](../../screenshots/transactions/01-image.png)](https://downloads.intercomcdn.com/i/o/1118938769/c4ce721c59c38ed52308798c/image.png?expires=1773322200&signature=416e50f095b60a60bfc32adfe05afd70570b6db70b4901de277d335b056fd05e&req=dSEmHsB9lYZZUPMW1HO4zaiBkIQtT02%2F%2F4YouJlzxKDC23qSKZR4xvibtiWW%0AK6hSRJCxkA1K92%2BLzDk%3D%0A)
***Tags*** can be used to associate any type of transaction, and do not have any budget associated with them. You can view Tag totals by month by filtering by one or more tags in the Transactions tab. Tags can be managed in several locations, including transaction lists, the transaction detail view, and in Settings under Features > Tags. **[Learn more about Tags here.](https://help.copilot.money/en/collections/9742377-tags)**
