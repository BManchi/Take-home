# Transactions Tab Overview

**Source:** https://help.copilot.money/en/articles/9554412-transactions-tab-overview

Copilot's Transactions tab allows you to review, filter, and export your transactions. You can also create new transactions, and edit the details of existing transactions.

---

# Search & Transaction Filters

There are a couple ways you can search for transactions in Copilot. You can type in the search field to find transactions containing a specific word or a specific merchant. Or, you can use transaction filters to find transactions that match one or multiple filters.

# Transaction Detail View

Transaction detail view allows you to view and edit existing transaction details like changing transaction date and name. Simply tap on the current transaction date or name to edit them. You can also update the [transaction type](https://help.copilot.money/en/articles/3971267-transaction-types) and its current [category](https://help.copilot.money/en/articles/3971270-transaction-name-rules-for-categorization).

You can also [split transaction](https://help.copilot.money/en/articles/5325255-splitting-a-transaction-across-multiple-months) and [mark a transaction as recurring](https://help.copilot.money/en/articles/3760068-create-recurrings) from this view.
[![screenshot](../../screenshots/transactions/05-image.png)](https://downloads.intercomcdn.com/i/o/1100663763/b34fcff673c325ab33d0de18/image.png?expires=1773322200&signature=40af1ba9ad7b493d940f4db3fe9f7226260450fdad2d2948d7700c91ced4db25&req=dSEnFs94noZZWvMW1HO4zbFrjH9TFcIKhHLblJPytINZcOC6bkKVEbbXDyEf%0AmWMLk3F%2FgF0njk1qDQU%3D%0A)
On **Copilot macOS and iPad**, transaction detail view has a few minor differences.

- The Mac and iPad app shows **Similar Transactions**, which are transactions with the same name and it shows the total monthly spending for these transactions as well.
- Currently, there is no option to view original transaction info.

# Add A New Transaction

You can use this to create manual transactions for any accounts. On **Copilot macOS and iPad**, you will click on the**+** symbol next to the search button. You can use the **[Creating Manual Transactions](https://help.copilot.money/en/articles/4038706-creating-manual-transactions)**article to learn more.
