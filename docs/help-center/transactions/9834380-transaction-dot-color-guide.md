# Transaction Dot Color Guide

**Source:** https://help.copilot.money/en/articles/9834380-transaction-dot-color-guide

The dots next to your transactions act as indicators.

**Gray** dots indicate a transaction that has an included tip amount.

**Blue**dots indicate transactions that need to be reviewed. These transactions can also be marked as reviewed in bulk.**[How to Bulk Edit Transactions](https://help.copilot.money/en/articles/7668990-bulk-editing-transactions).**

**Red** dots indicate that the current split doesn't sum to the total for the parent transaction if you have a split transaction. You can tap on the transaction details and SPLIT to edit the split and resolve the issue.
