# Tags

**Source:** https://help.copilot.money/en/articles/9554342-tags

You can use Tags to organize your transactions in a whole new way. With Tags, you can group together any type of transaction. There are a lot of ways to use this feature, including organizing your spending during a vacation, or for classifying different sources of income. Explore ways to get **[creative with tags here](https://help.copilot.money/en/articles/9554370-ways-to-use-tags)**.

Trying to understand the difference between tags and categories? **[Learn more here](https://help.copilot.money/en/articles/9554367-tags-vs-categories)**.

---

# An Overview

**In the Mac and iPad apps,** you can view all tags associated with a transaction by tapping on the tag chip. From here, you can add additional tags to the transaction or uncheck tags to remove them from the transaction.
[![screenshot](../../screenshots/transactions/01-image.png)](https://downloads.intercomcdn.com/i/o/1117937871/d72607ee09d81c813d9f53cc/image.png?expires=1773322200&signature=f3a9e860b1cc40d66bbbafc45b36cfa4604720836c4a760ede53ebaa67cfc2ce&req=dSEmEcB9molYWPMW1HO4zTVpNsEmZfEB2884IM6bdrEfcp1MZPvhiEVi9lMT%0AqONwNKic1iU0lxwLEtI%3D%0A)
**In the iOS app,** you can simply tap on a transaction to see all tags associated with a transaction in the transaction detail view. From here, you can add additional tags to the transaction or remove existing tags.
[![screenshot](../../screenshots/transactions/02-image.png)](https://downloads.intercomcdn.com/i/o/1117977983/bbf890f258b0c04761184f1b/image.png?expires=1773322200&signature=1878e95614684c27cff752b3b99487ac75a15798912c09b3f4b1370d83c100a7&req=dSEmEcB5mohXWvMW1HO4zXtsSUZgxVjVTUtGPkPLovi02u8nHA6FN0vvj3BA%0A5cayg6dqnaDoajI30cQ%3D%0A)

# Creating and Managing Tags

You can create a new tag from multiple locations, including while tagging a transaction. The new tag will automatically be applied to the transaction after creation:
[![screenshot](../../screenshots/transactions/03-image.png)](https://downloads.intercomcdn.com/i/o/1117978487/2bf650a0a17dbb0ad3380a20/image.png?expires=1773322200&signature=391f3a74df152658dae60124a815b436dc1e25f1921c865157c46ac8f142996f&req=dSEmEcB5lYVXXvMW1HO4zS0eyZAutc%2FxqBOtxCgaaqABzu9T8G3ywFF4kWXt%0AY2g6%2F2JD4QG300NXEmg%3D%0A)
And from the new Tags section in Settings:
[![screenshot](../../screenshots/transactions/04-image.png)](https://downloads.intercomcdn.com/i/o/1117937113/9f117cbe3b9ad6887a4d8e58/image.png?expires=1773322200&signature=2fe8b36d92143099ad35a452bba042b8f3c459a7bf2a6ce2b65f1bdc588d5c61&req=dSEmEcB9moBeWvMW1HO4zW3HSp9oBXTEJWVPhkSLlVKBjfR8%2F8V%2FgJquSav5%0AXKHLVe9gAYO8MdRVoMw%3D%0A)
Please note that you can’t create a tag with the same name as an existing tag.

In the Tag Management section, you’ll be able to view a full list of your tags and edit them. Tags are sorted alphabetically in this list (capitalized, then lowercase).

# Bulk Applying Tags

In the iOS, iPad, and Mac apps, you can bulk apply tags.
[![screenshot](../../screenshots/transactions/05-image.png)](https://downloads.intercomcdn.com/i/o/1117984774/8623f71b85fe128fd968de07/image.png?expires=1773322200&signature=367373a67ee03b5efea06446106e2fc4c0ff92da105a7af99d5423d5e8e7d6dc&req=dSEmEcB2mYZYXfMW1HO4zVG76d94RdKV3ICnH2t%2BTjdpYvaDa8q2DE7mnSNy%0Am%2BcVn4RDIZ%2FjvL%2Ft7SM%3D%0A)
If there is a line through the check field in bulk editing, it means that some of the selected transactions already have that tag applied. You can tap to unselect the tag for all, and then re-select to apply the tag to all selected transactions.

# Filtering by Tags

To see monthly totals, simply filter the Transactions tab by one or more tags.
[![screenshot](../../screenshots/transactions/06-image.png)](https://downloads.intercomcdn.com/i/o/1117979224/8ff4ff697f9ad9ef87006891/image.png?expires=1773322200&signature=ae871165ae99cc724074d72f3b31abb206d73152b248c315474d655b426d0f4c&req=dSEmEcB5lINdXfMW1HO4zSO7ojlr2UibDRmwC3CGi5WhUX1nNqLdh6JG6Cbt%0APJMob6BNqaPDzqZnRGM%3D%0A)
You can also use the search bar to search tag names and select them for filtering.

---

# Adding Tags to a Manual Transaction

At this time, you can only add tags to manual transactions after creating the transaction. Once you’ve saved the new manual transaction, you can add and create tags from the transaction detail view.
