# Creative Ways to Use Tags

**Source:** https://help.copilot.money/en/articles/9554370-creative-ways-to-use-tags

There are a lot of ways to use tags, including organizing your spending during a vacation, or for classifying different sources of income. Below are some ideas, sourced from the Copilot team, on how to use tags.

**Different types of income:**Because tags can be used for any type of transaction, you can use tags to differentiate between different types of income.
[![screenshot](../../screenshots/transactions/01-image.png)](https://downloads.intercomcdn.com/i/o/1118880106/06c0ef19772706c4ec3fa12c/image.png?expires=1773322200&signature=8d1b93b0260580cef24dd6a0fff956d15a7b9329bd1f8c37d8d0a71970505bba&req=dSEmHsF2nYBfX%2FMW1HO4zfhDycJwcwBtLphQ9SpHi0XW7A4qwn9GPaBKfVFF%0AjY6%2Bjnm2%2F9Ex1LhmhoU%3D%0A)
**Trip spending:** Tag transactions with a trip specific tag to track the total spent for a trip per month.
[![screenshot](../../screenshots/transactions/02-image.png)](https://downloads.intercomcdn.com/i/o/1118878806/483856feda2aa63dcd3ad4c3/image.png?expires=1773322200&signature=68d294ba1159e43844a731fdbfb74d055d19eec048aa0e840065efbd29c25771&req=dSEmHsF5lYlfX%2FMW1HO4zZExSrhKxUaAdgPbY42R%2BOqI0D%2BmTP%2FrF7cyWXqp%0Atj4BSO%2BPRd0rlZyWLN8%3D%0A)
**Projects or home improvement:** Tag transactions with a specific tag to track the total spend on a specific project per month.
[![screenshot](../../screenshots/transactions/03-image.png)](https://downloads.intercomcdn.com/i/o/1118880493/8199ed74ae35ce88634afe38/image.png?expires=1773322200&signature=d8be8542dd434e4e4e5e0f36f3ab5b84c6476ef6055b6a994ec6f58683fc7d8a&req=dSEmHsF2nYVWWvMW1HO4zcutgRrtfQqO6sRq2R1YjlKXP6yLuNCJqi9P7uJ8%0AuUuECebm331ME1zAR8A%3D%0A)
**Family member specific spend:** Tag transactions with a specific tag to track spend on specific individuals across multiple categories.
[![screenshot](../../screenshots/transactions/04-image.png)](https://downloads.intercomcdn.com/i/o/1118879288/bce162b2742bdb4c093774af/image.png?expires=1773322200&signature=b5ba1748fe2c9a96674a831b964d0727ca4f1969c310baa2ffbe0a01ce257df7&req=dSEmHsF5lINXUfMW1HO4zRaEcr%2Fh1fhqJz%2FPx4RBF2hfUd2kvSL%2Fu0hN6eB6%0AOm3%2F0l%2FfnHkt52ACCdc%3D%0A)
**Transactions awaiting refunds:**Tag transactions that you’re awaiting refunds or repayment for (in the case of splitting a meal with friends!) to make sure you don’t forget to follow up on anything.
[![screenshot](../../screenshots/transactions/05-image.png)](https://downloads.intercomcdn.com/i/o/1118880870/f81abbae7f41ed91529d2802/image.png?expires=1773322200&signature=084d479c06abe7a9a9e3e703bfdbab06595d00e29b070c5a37c0a87667ebecd0&req=dSEmHsF2nYlYWfMW1HO4zYyltk6d2QMmg05Tg3Np5PBDzos0D7D4oNiUv%2F6i%0AXb2iJfVMXF%2F6LNlI9bw%3D%0A)
**Transactions to review again, or with a partner:** Flag transactions you’d like to review again later without leaving them in the “To Review” state.
[![screenshot](../../screenshots/transactions/06-image.png)](https://downloads.intercomcdn.com/i/o/1118879637/1b01a573ac272fb81432671c/image.png?expires=1773322200&signature=cc59d14a4f6bee7cb22eb057f8148cba55390c6b2235e27aee28c3a7f70ac243&req=dSEmHsF5lIdcXvMW1HO4zYtmvTvlJUZCwwQeOPbGsmPFSFSwer7W2oRRt15K%0Asv1sc%2FuaPiBp8n1padQ%3D%0A)
