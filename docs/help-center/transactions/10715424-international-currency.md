# International Support

At this time, we only support US financial institutions and as such are only available in the US App Store; however, we would like to expand our coverage in the future.

You can upvote this feature request [here.](https://copilot.canny.io/feature-requests/p/international-support)

# Converting Foreign Currency

If you have an account that requires currency conversion, please note that Copilot only supports USD and does not convert foreign currency transactions. Any transactions in other currencies will appear in USD based on the amount received from the data provider.

As a workaround, if you have expenses in other currencies, you can manually edit a transaction amount to reflect the converted amount in USD. Simply tap on the **Transaction > Tap the current amount > Edit > Save.**
