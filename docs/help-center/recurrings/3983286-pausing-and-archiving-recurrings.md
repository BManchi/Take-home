# Pausing and Archiving Recurrings

**Source:** https://help.copilot.money/en/articles/3983286-pausing-and-archiving-recurrings

# **Pause a Recurring**

Copilot allows you to pause a Recurring if the expense has been temporarily paused in your budget. For example, if you want to take a break from TV, you may pause your streaming service subscription for three months.

To **Pause** a Recurring, go to the **Recurring** view and select it.

Once you have paused a Recurring, Copilot doesn’t expect that Recurring transaction to be paid in the future, so the expense will be removed from your expected spend if it hasn't been paid already.

The next time Copilot receives a payment for a paused Recurring, it will automatically be marked as active again.

# **Archive a Recurring**

Copilot also allows you to archive a Recurring if the expense has been permanently removed from your budget. For example, if you cancel a gym membership, then you can archive the Recurring transaction data for historical reference.

To **Archive** a Recurring, go to the **Recurring** view and select it.

# **Activate a Recurring**

Copilot also allows you to activate a Recurring that was previously marked as **Paused** or **Archived**.

To **Activate** a Recurring, go to the **Recurring** view and select it.

# **Recurring View Options**

Copilot on iPhone allows you to view your Active (Monthly & Future), Paused and Archived Recurrings in either a block or list view

To change the **Recurring** view, tap the block or list icons on the right hand side of each section.
