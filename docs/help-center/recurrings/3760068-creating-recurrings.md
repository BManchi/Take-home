# Creating Recurrings

**Source:** https://help.copilot.money/en/articles/3760068-creating-recurrings

Copilot’s Recurring feature allows you to automatically capture your expected spend, such as Rent, Bills, and Subscriptions. You can create recurring schedules for weekly, bi-weekly, monthly or non-monthly spend.

---

# Create a Monthly Recurring from the Recurring View

Tap the **+**button to create a new schedule. Then, search for the recurring transaction by name.
[![screenshot](../../screenshots/recurrings/01-Pic01.png)](https://downloads.intercomcdn.com/i/o/525554742/d0cd35889f80dd39160437ca/Pic01.png?expires=1773322200&signature=eff9953e5a68e24ebf0df1fbfba011cef17bfe3232e234fb52ac52ee33c1f792&req=cSIiE8x6moVdFb4f3HP0gCBVCO68%2FDYosBkS9NHcEgxLGYuHUq6dl1XHXdmR%0ATeW%2BEF%2B0bb%2FfeSiCGg%3D%3D%0A)
Select either an individual transaction or create a recurring for all transactions that contain that transaction name. We will estimate the frequency based on the transaction(s) selected in the step above. **Confirm the Frequency**.
[![screenshot](../../screenshots/recurrings/02-Pic02.png)](https://downloads.intercomcdn.com/i/o/525554774/d31744f0096855a5846e3f02/Pic02.png?expires=1773322200&signature=271182a4a399c4fd500912c11d9dca2825c88ded7b3c3015ee13cc6a4d4a85e0&req=cSIiE8x6moZbFb4f3HP0gKvN%2BVcxTsaBKrYtLsd8BAWu%2BvFxA%2BiJvl4sGjGf%0AajPXrhsPIE0WpB8qjQ%3D%3D%0A)
For this example, select **Every month** to open the **New Monthly Recurring** view.
[![screenshot](../../screenshots/recurrings/03-Pic03.png)](https://downloads.intercomcdn.com/i/o/525554804/435703f8702495573a8dc584/Pic03.png?expires=1773322200&signature=3e6aa348750b79e9eef86da79b44ed1d5244101567b05febeca23bd609fa27c4&req=cSIiE8x6lYFbFb4f3HP0gM09LbpffR2FKqlzH6sozRB5HG8ZcPykUwMRj3mo%0AXoAPiZiLWaA5GW9Kew%3D%3D%0A)
In the **New Monthly Recurring** view, you can edit the emoji, name, filter settings (ie. Transactions named *Spotify* from *$9* to *$10*around*the 23rd*every*month*), category and transactions (add or delete).
[![screenshot](../../screenshots/recurrings/04-Pic04.png)](https://downloads.intercomcdn.com/i/o/525554843/1eea5271283ddfdd14fbfa71/Pic04.png?expires=1773322200&signature=9dcf4dd283e60c152ff359226a635910e7ee575cba46a6cd66a4dbd187b18a43&req=cSIiE8x6lYVcFb4f3HP0gFwzo3aNy3KTcPOxQ4joXEM6k4eoW%2B%2BybsuDMD%2Bp%0AXyHrgv92JG07b9JC1g%3D%3D%0A)
After you save the **New Monthly Recurring**, your next expected payment will be noted and the transaction(s) will be marked with an **[R]**to indicate **Recurring**.
[![screenshot](../../screenshots/recurrings/05-Pic05.png)](https://downloads.intercomcdn.com/i/o/525554922/a6e9f0ed939155fecab34005/Pic05.png?expires=1773322200&signature=0cb5ac144428fcf8b1c6907ec1d4d39ea799a53814b66498123fb3fd97a0fcf6&req=cSIiE8x6lINdFb4f3HP0gKYTv2W5ff1eW%2FwlzjMJ627iF3YeQLCdgv%2Fv9v8E%0AhVoZR47Q%2BPc9IywApw%3D%3D%0A)[![screenshot](../../screenshots/recurrings/06-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1211692422/95604f8f70954d24e71269aefd2d/image.png?expires=1773322200&signature=7d84ed7101644e35f106fa3917965141503e26cd4bffb13b249e4cf1b98023bb&req=dSImF893n4VdW%2FMW1HO4zUQbD2g5714L3sNerNTQyMsaiP545tQUdf3vvk0T%0AN5myAe7iqTdRsvFgd0E%3D%0A)
# Create a Monthly Recurring from the Transactions View

Search for the recurring transaction by name.
[![screenshot](../../screenshots/recurrings/07-Pic06.png)](https://downloads.intercomcdn.com/i/o/525554967/dae7e21cf0fddbc254f640e4/Pic06.png?expires=1773322200&signature=0c2ee6ddcef3eec9206eeb2034d1808585c21f75df5ebc555d211e0c181a9555&req=cSIiE8x6lIdYFb4f3HP0gI6BoSaUwV45TQvXjvYGWhuyMt%2FaXwnKomgSUxuv%0AejR0HNRfIvybhDiq3g%3D%3D%0A)
Select one of the transactions and select **Recurring** at the bottom of the view.
[![screenshot](../../screenshots/recurrings/08-Pic07.png)](https://downloads.intercomcdn.com/i/o/525555020/0ec6b84f96013efd8b8184d5/Pic07.png?expires=1773322200&signature=af1fc917123af53e4515b09d801a2c5a057b7b0a5a20d68b17462d736ab7b1c8&req=cSIiE8x7nYNfFb4f3HP0gMpZiimQLl6ePGBNFp7qVVhIK6aFRi6EqPWVBbah%0AD%2BtTZIr7buNvi5XVBQ%3D%3D%0A)
The**Recurrings** menu will open with a list of your existing recurrings, plus the option to **Start a new one**.
[![screenshot](../../screenshots/recurrings/09-Pic08.png)](https://downloads.intercomcdn.com/i/o/525555050/194a130eeadea9a765fdca20/Pic08.png?expires=1773322200&signature=d7b8611a7db8af97722a59a7f49e8115ac09c681a8f93198bd30c5374781401e&req=cSIiE8x7nYRfFb4f3HP0gESRR5BU4phfZWBJDmtMtnrUXohIlhTXjqqJKbMj%0A89DBvzKD4x665IzLGw%3D%3D%0A)
After you select **Start a new one**, the **New Monthly Recurring** view will open. Here you can edit the emoji, name, filter settings (ie. Transactions named *Spotify* from *$9* to *$10*around*the 23rd and 24th*), category and transactions (add or delete).
[![screenshot](../../screenshots/recurrings/10-Pic09.png)](https://downloads.intercomcdn.com/i/o/525555080/b8733763d26cfc5c0def2273/Pic09.png?expires=1773322200&signature=0cda963b9c18c9f5535e4b8cec4f43f17c163d98eaaa70d5c31d3974000e1325&req=cSIiE8x7nYlfFb4f3HP0gG1TdtZdteiU0HSySHC4MjVxnZG%2B7Sq73rwuexI5%0AMCL9wxLGZNFiRKZKFw%3D%3D%0A)
After you save the **New Monthly Recurring**, your next expected payment will be noted and the transaction(s) will be marked with an **[R]**to indicate **Recurring**.

# Create a Non-Monthly Recurring from the Recurring View

Tap the **+**button to create a new schedule. Then, search for the recurring transaction by name.
[![screenshot](../../screenshots/recurrings/11-Pic10.png)](https://downloads.intercomcdn.com/i/o/525555120/ec3e685898723f00e3e689ff/Pic10.png?expires=1773322200&signature=7b8548d57061a7d1f146999cb468a89648beaae619c1c921e926f322913620ce&req=cSIiE8x7nINfFb4f3HP0gNaDUdnF%2FKk5hpJJUpB6a%2F4sdOvMd0T3r%2FwWGAyJ%0ASkR1%2FEg%2FFPti%2F5Y5IA%3D%3D%0A)
Select either an individual transaction or create a recurring for all transactions that contain that transaction name. We will estimate the frequency based on the transaction(s) selected in the step above. **Confirm the Frequency**.
[![screenshot](../../screenshots/recurrings/12-Pic11.png)](https://downloads.intercomcdn.com/i/o/525555145/94f748daafc20a7a6a83f994/Pic11.png?expires=1773322200&signature=16be0ac8512a3f95313f19dbc24015e6ad46dd0ce6d32f205eaa4a0b6126296b&req=cSIiE8x7nIVaFb4f3HP0gEeGQ%2FEnJmII4LqNY3XyUCSfyVZHDlH7kZhgUKoD%0Avgz5%2FT6VaWgHUII6Wg%3D%3D%0A)

In the **New Annually Recurring** view, you can edit the emoji, name, filter settings (ie. Transactions named *Gaia* from *$95* to *$96*around*the 19th and 20th*every*year*), category and transactions (add or delete).
