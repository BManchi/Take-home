# Recurrings FAQ

**Source:** https://help.copilot.money/en/articles/10244751-recurrings-faq

## Why isn't my transaction automatically linking with the Recurring I created?

To resolve this tap on the transaction > tap RECURRING at the bottom of the view to associate it with the Recurring. Then you can review your filters to make sure they are optimized to catch future transactions.

**[Optimize Recurrings](https://intercom.help/copilotmoney/en/articles/3783499-optimize-recurrings)**

## What if I have multiple Recurrings with the same vendor?

If you have more than one recurring posted under the same merchant name, then you should create multiple Recurrings.

**[Multiple Recurrings for the Same Merchant](https://intercom.help/copilotmoney/en/articles/5327632-multiple-recurrings-for-the-same-merchant)**

## Why is my recurring showing as spent when the transaction has not happened?

Recurrings are considered fixed expenses so they are already counted as spent even if the transaction has not occurred yet.

## Can I create a Recurring for a transaction that hasn’t happened yet?

A recurring can only be created from an existing transaction in the Copilot Money app.

## How do I create recurring income?

Recurring income is not supported and can only be created with regular transactions.
