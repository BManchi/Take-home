# Creating New Goals

**Source:** https://help.copilot.money/en/articles/11100462-creating-new-goals

You can create a new goal directly within the Goals tab or from an existing transaction. To create a new goal from the Goals tab **in the Mac and iPad apps**, you can tap on **Add a goal** in the top header bar to create a new savings goal.
[![screenshot](../../screenshots/goals/01-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532883387/7ddd7655f6b2ab69f4faaeabdbbd/image.png?expires=1773322200&signature=54437c45594ca252a14ed88fe6ac9d7df543fbef7cdbb85adac90a85b14abece&req=dSUkFMF2noJXXvMW1HO4zdhjGCMThgYuoIAAE7SFtMKAyndLhL7janJEIbZ6%0A9fPxwzZot7CpwwH4im0%3D%0A)
**In the iOS app**, you can select Add a goal at the bottom of the Goals tab.
​
[![screenshot](../../screenshots/goals/02-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1545325418/6b43abac97f1da3ff7aba45e0755/image.png?expires=1773322200&signature=370b2b0353eed0ca422d747bd77eb10250c03262be08032672091a10668b9088&req=dSUjE8p8mIVeUfMW1HO4zZmjjk0KNtClH%2FgCeUA15sj8wfus63zx0VOsfbUe%0AruRuEVvh9K2FBULC1WU%3D%0A)
To create a savings goal from an existing transaction, tap on a **transaction > Goal > New** to create a new savings goal.
[![screenshot](../../screenshots/goals/03-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532883683/a8f3add821469d59e8779e29520a/image.png?expires=1773322200&signature=4c4da0b9f052594dcbac1561f4db168229c635667352af6fc9d7a0927c1de360&req=dSUkFMF2nodXWvMW1HO4zVJuZ7L0BAf%2FUx8p8r5WcwBvzoRaWX3PwjyvSVli%0AjjMBUKFYpf9CJiokx6E%3D%0A)
Both of these actions will bring you to the same goal creation screen. Then, you'll decide what you want to save for! You can use any of the recommended goal listed, or you can create a custom goal. If you choose to create a custom goal, you'll be prompted to give the goal a name and an emoji before continuing.
[![screenshot](../../screenshots/goals/04-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532886054/6d1f17608b4294951fdc5a4aeac8/image.png?expires=1773322200&signature=b2ee115f8eb65959efb782ada71d2f594b40d32db18337357d63401858e56d78&req=dSUkFMF2m4FaXfMW1HO4zZCM38nQOr%2FrSC70%2B9%2FFR%2BZWM7dzL8n5Y8O1cLLJ%0AN7T4XYinTtZ0%2FwOiXs4%3D%0A)
Then, enter the target amount you want to save for and tap continue.
[![screenshot](../../screenshots/goals/05-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532886534/13496269f6d4fdfa4321f0cf22cc/image.png?expires=1773322200&signature=70dec43a3991e4c6f1fda5a4795c5264b5a6dbd8ec6c29fe707694bcd5aa5435&req=dSUkFMF2m4RcXfMW1HO4zZLQz3sqTOEXarcArg%2FOiuMT3GJZBr213D8JiUpl%0AKaFIQqBF9iVzl2W7hdM%3D%0A)
Next, you'll decide if you want to start a goal from scratch or if you already have savings from account(s) that can be use for this goal.
[![screenshot](../../screenshots/goals/06-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532892197/e00f144c190bdb40a9dd763dfc0e/image.png?expires=1773322200&signature=8b79d6fe8a7c7518150cf3f2d4329599459517be9ba2e843b7047da8bcefef38&req=dSUkFMF3n4BWXvMW1HO4zWBN4jpM783oYSMIQFLVjMb%2BcaebRq8d50XautQl%0AZDBBb7Toh18Ho68wWFA%3D%0A)
If you already have savings that can be use for this goal, you'll be prompted to allocate part of the account's balance for this goal by dragging the balance bar or tapping to enter an amount. Or, you can toggle on **Always use the entire balance** to use the entire balance of the selected account.
[![screenshot](../../screenshots/goals/07-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1534590177/a88583cdff8d61ce98d5c8d2832b/image.png?expires=1773322200&signature=ddd7a0c2016068d00682acd785f7615c87a8cda25f24f1b420b0e987fd1eabe8&req=dSUkEsx3nYBYXvMW1HO4zfHYA%2F0GbiRZ12SJ3tB8s2Op4w9naFJGjKN%2FTrbz%0ALGzoXI4x%2B8YripelA%2Bk%3D%0A)
Next, pick how you would like to track the progress of the goal. Either by selecting a **target date** or **monthly contribution**. This can also be edited at a later time.
[![screenshot](../../screenshots/goals/08-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532892497/82ed269dc03d787bcce4a2715b32/image.png?expires=1773322200&signature=a3e5b24e08ad2e0cabdb3f1a3bfb97fd5b0fdd7e442238d4264f10bdfbf182c6&req=dSUkFMF3n4VWXvMW1HO4zT%2FpihHs0P7yYpdsruI3KP5pT7Eyq3zLMHlIARjR%0ADcSpsSc%2F4LhakAj%2FoCA%3D%0A)
With **target date**, you'll decide when you want to complete this goal by and we'll show you the amount you'll need to save every month to complete this goal in time.

***Note:****The last day of the target month is the last day to complete the goal in time. So if you want a goal to be ready on the 1st of a particular month, the target month should be set to the month before.*
[![screenshot](../../screenshots/goals/09-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1543671053/5921ad213c0687459e27d7caa188/image.png?expires=1773322200&signature=e10b291308a5267911d82583cf38ab0b945c75a6aae71aced199c768350cf7cf&req=dSUjFc95nIFaWvMW1HO4za7N1kmQ%2BXTVXIjdD4dc3OUyQ6fhFCvYV6x30pJp%0AqINMxwjBvasYY5sVUL8%3D%0A)
With **monthly contribution**, you'll decide how much you want to save each month, and we'll show you how long it'll take to reach the goal.
[![screenshot](../../screenshots/goals/10-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1543670286/925641c48cb5fbe04aacb4c0da2e/image.png?expires=1773322200&signature=4b44dd2fdf4f66c76265c59a9788564417de5200039a3a14e906df87f007c8a8&req=dSUjFc95nYNXX%2FMW1HO4zdmpd1K6L5XAjHBJ4uRhbkCYC%2BE1ik7ciJAlktEn%0AVXQ%2BuB%2FLL%2Bw3S6xGMNY%3D%0A)
Then, review all the selections you have made for this goal and tap **Save goal** to create the goal. You can also edit all the settings listed within the review screen to make any adjustments if needed. You can also edit the contributions before saving the goal as well.
[![screenshot](../../screenshots/goals/11-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532895371/40452b027bd315756e1b39457cf6/image.png?expires=1773322200&signature=9fb4ebd565d23ee838f0025087ae65d8cf22411aa47ecfd0f46cbe09c126e5fd&req=dSUkFMF3mIJYWPMW1HO4zeHvC3JbHQwMLM7McRehLo%2BBTf%2BRyXZaxZFK2OVO%0ALIa3B1kcRwk%2BSeSbZ0Q%3D%0A)
You can update the Goal settings for an active goal at any time by tapping on the goal, then tap on Goal settings, then you can tap on any of the settings to update the selection.
[![screenshot](../../screenshots/goals/12-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1543676328/e6ab451c5f39d0ff66fc429cdda2/image.png?expires=1773322200&signature=00578b2e78fadd58ad7d77213cadf5b51d59fc4cec395c87be3b2fd55d2de7dd&req=dSUjFc95m4JdUfMW1HO4zUMTbpxQHRqWDbr6lFA2L1fGwZ3pMgDsMrkr%2F3mM%0ASCZ%2FEXpamoQPONboInM%3D%0A)
On the **macOS app**, you can tap on an active goal then tap on any of these settings in the Summary section to make adjustments.
[![screenshot](../../screenshots/goals/13-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1543675919/7b45d7f6157ffca3c609764ee00e/image.png?expires=1773322200&signature=45711ef092e4c070c1a94b8010938b05614affb02888ffaafb02eec03618f354&req=dSUjFc95mIheUPMW1HO4zd8G5T%2FQwmu7DACEkHVYk63JKSj2suUsHENFKT2s%0A3DoivbJ58e8KG0QhU0g%3D%0A)
