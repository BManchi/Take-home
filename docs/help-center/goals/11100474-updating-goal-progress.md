# Updating Goal Progress

**Source:** https://help.copilot.money/en/articles/11100474-updating-goal-progress

You can update goal progress by editing the allocation or adding transactions.

---

# Update Progress by Editing Allocation

Within the Goal details, tap on **Update Progress**, then tap on **Edit allocations**. Then, you'll see a list of depository accounts you have, including hidden accounts. From there, you can tap on the current allocated amount to update the allocation you want to assign to this goal, then tap continue to save the change.
[![screenshot](../../screenshots/goals/01-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532940335/9f3bd3cb6c3b4dde075cfc584a3e/image.png?expires=1773322200&signature=c374d47d6591789ebaaa441920e151af5b9154bccff12a93fb7e64be98fa5784&req=dSUkFMB6nYJcXPMW1HO4zV1N%2FrOQ331E7JW6zswWimexhjMcyRROrInnZdgT%0A9S1dXUcLKaCQ3N0EBrY%3D%0A)
If **Always use the entire balance** setting is toggled on for an account, the balance change in this account will impact the amount saved for a goal automatically.

# Update Progress by Adding Transactions

Within the Goal details, tap on **Update Progress**, than tap on **Add transactions**. Then, you'll see a list of transactions that you can assign to a goal. Or, search for a transaction using the Search bar.
[![screenshot](../../screenshots/goals/02-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1543687801/c8f585cbefdc9a875024a4197ac6/image.png?expires=1773322200&signature=6cadb5e4e813e54d49dedb4ee4501355cc89fd4a1b12804aed254e014693d8ec&req=dSUjFc92molfWPMW1HO4zXRKUlGZrhTnN%2F%2BEuc6M1SXsrpZ4%2FTtYnhVJPcSs%0AN%2F7LMXFE3sVpCkLBrJ4%3D%0A)
Learn more about **[how spending from an active goal can impact your category and budget here](https://help.copilot.money/en/articles/11100511-spending-from-savings-goals#h_f9c75fdfc4)**.

When adding an internal transfer transaction to a goal, you'll be prompted to confirm which side of the transfer you want to associate with the goal, outgoing or incoming.

If you want the transaction to **decrease** the amount saved of the goal, you'll want to assign the outgoing transaction to the goal. If you want the transaction to **increase** the amount saved of the goal, you'll want to assign the incoming transaction to the goal.
[![screenshot](../../screenshots/goals/03-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1543694254/524103ba95353a60daef900cf72e/image.png?expires=1773322200&signature=2640d5811e7a734d8de88c5bf5882be3a6e578c1bb00d52444707038d9e028cc&req=dSUjFc93mYNaXfMW1HO4zTW1hmL2oR1S1ni4LFFwGeavj1f4SoaSiT62cmdX%0A5gUxRj2ls92KVB%2BaSPw%3D%0A)
**Note:**If the incoming ("to") transaction is unavailable for your transaction, due to the incoming account being an investment or manual account, you can tap "select an account" for Copilot to auto-create the necessary transaction to allocate funds towards your goal progress.

# Completing a Goal

When the goal reached its targeted amount, it will be automatically marked as complete and moved to the **Ready to Spend** section to indicate that you can start spending from the goal.
[![screenshot](../../screenshots/goals/05-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532948346/078830246744938dfeffbe1f5e56/image.png?expires=1773322200&signature=10221660f74facf3ba1e8441d2ce26c69bae827d8366fc62c7c7d1cf3f3ad94c&req=dSUkFMB6lYJbX%2FMW1HO4zUZ35MT%2Bv1OCP4AE%2BwYylqDN8rSavYQM9jE%2FUmkL%0A4FQJYUyElSkbeAemNWs%3D%0A)
If **Reactive when funds dip** setting is toggled on for this goal, the goal will be moved back to **Active** as you spend from this goal by associating transactions with this goal or by editing allocations. If the setting is toggled off, the goal will stay in **Ready to Spend**.
[![screenshot](../../screenshots/goals/06-image.png)](https://downloads.intercomcdn.com/i/o/gw2wbwl7/1532948021/66ff2d81e487e14004d57a6c8d60/image.png?expires=1773322200&signature=3e95c81797eeebb58e84c37e208348522adc8ff83f0413a5821d21f8c79e984a&req=dSUkFMB6lYFdWPMW1HO4zUqpAtK3a%2B9p81OwatfwkcL28YTuSJSCt6cSRc89%0AT8b%2FeE2R0s9Aa0C6MCM%3D%0A)
# Completing a Goal Early

You have the option to complete a goal early as well, this means that we'll adjust the targeted amount to whatever amount you have already saved for the goal and move the goal into **Ready to Spend**.
